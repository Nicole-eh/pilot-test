# 🎓 Node.js 初级功能学习指南

## ✅ 已实现的三个初级功能

### 1️⃣ 命令行参数处理

**功能说明**：程序可以接收用户输入的名字并进行个性化问候。

**使用方法**：
```bash
node index.js 小明
node index.js Alice
node index.js 张三 李四  # 支持多个词
```

**输出示例**：
```
你好, 小明! 👋
✓ 日志已保存到 log.txt
```

**学习要点**：
- `process.argv` 数组的使用
- `process.argv[0]` 是 node 的路径
- `process.argv[1]` 是脚本文件路径
- `process.argv.slice(2)` 获取用户输入的参数

---

### 2️⃣ 文件读写（日志功能）

**功能说明**：自动记录所有操作到 `log.txt` 文件，并可以查看历史记录。

**写入日志**：
```bash
node index.js log 今天学习了 Node.js
node index.js log 完成了计算器功能
```

**查看日志**：
```bash
node index.js logs
```

**输出示例**：
```
📋 历史日志：
─────────────────────────────────────
[2026/03/24 11:46:47] 今天学习了Node.js的基础功能
[2026/03/24 11:46:47] 计算: 5 + 3 = 8
─────────────────────────────────────
```

**学习要点**：
- `fs.appendFileSync()` - 追加内容到文件
- `fs.readFileSync()` - 读取文件内容
- `fs.existsSync()` - 检查文件是否存在
- `path.join()` - 拼接文件路径
- `try...catch` - 错误处理

---

### 3️⃣ 简单计算器

**功能说明**：支持加减乘除四种基本运算，并有完善的错误处理。

**使用方法**：
```bash
node index.js add 5 3        # 加法
node index.js subtract 10 4  # 减法
node index.js multiply 6 7   # 乘法
node index.js divide 20 4    # 除法

# 也支持中文命令！
node index.js 加 5 3
node index.js 乘 6 7
```

**输出示例**：
```bash
5 + 3 = 8
✓ 日志已保存到 log.txt
```

**错误处理**：
```bash
node index.js divide 10 0
# 输出: ✗ 错误：除数不能为 0！

node index.js add abc 5
# 输出: ✗ 错误：请输入有效的数字！
```

**学习要点**：
- `parseFloat()` - 将字符串转换为数字
- `isNaN()` - 检查是否为有效数字
- `switch...case` - 多条件判断
- 输入验证和错误处理

if (require.main === module) {

## 🐛 常见问题

**Q: 为什么运行后没有 log.txt 文件？**
A: 检查是否有写入权限，文件会自动创建在脚本所在目录。

**Q: 计算结果不对？**
A: 确保输入的是数字，不要包含其他字符。

**Q: 中文乱码怎么办？**
A: 确保终端支持 UTF-8 编码。
